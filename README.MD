# Desafio FullCycle rate limiter

Este projeto implementa um middleware de rate limiting baseado em IP e tokens de acesso, utilizando Redis como armazenamento.

## ⚙️ Tecnologias

- Go
- Chi
- Redis
- Docker / Docker Compose
